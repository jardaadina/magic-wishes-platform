# MagicWishes Platform

MagicWishes is an online platform that helps users create personalized wishlists and participate in Secret Santa events. Through this platform, friends and family can easily discover what their loved ones want and contribute to fulfilling other users' wishes.

## Key Features
* **User Management:** Create and manage user accounts.
* **Profile:** Secure authentication and profile editing.
* **Wishlists:** Add, view, and manage personalized wishlists.
* **Secret Santa:** Participate in Secret Santa events.
* **Public Access:** Visitors can view public wishes without logging in.
* **Fulfill Wishes:** Users can mark wishes as fulfilled for others.

## User Roles (Actors)

### Authorized User (Client)
* Can add and manage their own wishes.
* Can participate in Secret Santa groups.
* Can interact with other users and fulfill wishes.

### Visitor
* Can view public wishes.
* Can explore the platform.
* Has the option to register to access full features.
